1. Text Parsing

A walk through a stream of bits to identify a pattern

1.1. Example of text pattern

abcde

1.2. What do I want to represent

abc,de

1.3. How can we parse that

1.3.1. Skeleton

class parser:
  def __init__(self, code):
    self.code = code
    self.pos = 0

  def curC(self):
    if self.pos >= len(self.code): return None
    return self.code[self.pos]

  def nextC(self):
    self.pos += 1

  def matchC(self, c):
    if self.curC() == c:
        self.nextC()
        return True
    return False

1.3.2. Read the bits

def parse(self):
  newList = []
  while self.curC():
    if self.curC().isalpha():
      d = self.pos
      while self.curC() and self.curC().isalpha():
        self.nextC()
        newList.append(self.code[d:self.pos])
    elif self.curC() == ',':
      self.nextC()
    else:
      raise SyntaxError("Char %s isn't expected" % self.curC())
  return newList

1.4. Voilá

>>> parser("ab,c,d").parse()
['ab', 'c', 'd']

def parseList(self):
